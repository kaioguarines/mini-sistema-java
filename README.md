# Mini Sistema em Java

Aplicação desenvolvida para praticar lógica de programação utilizando Java.

## 🚀 Funcionalidades

- Multiplicação de dois números
- Verificação se o número é par ou ímpar
- Gerador de tabuada

## 🛠 Tecnologias utilizadas

- Java

## 📚 Conceitos aplicados

- Métodos
- Estrutura condicional (if/else)
- Operador módulo (%)
- Estrutura de repetição (for)

## 🎯 Objetivo

Projeto criado com foco no desenvolvimento da lógica de programação e consolidação dos fundamentos da linguagem Java.
