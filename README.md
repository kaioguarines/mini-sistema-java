# 🧮 Mini Sistema em Java

Aplicação desenvolvida em Java com foco na prática de lógica de programação e consolidação dos fundamentos da linguagem.

---

## 🚀 Funcionalidades

- ✔ Multiplicação de dois números
- ✔ Verificação se um número é par ou ímpar
- ✔ Geração de tabuada

---

## 🛠 Tecnologias Utilizadas

- Java 17+
- IntelliJ IDEA
- Git
- GitHub

---

## 📚 Conceitos Aplicados

- Métodos
- Estruturas condicionais (if/else)
- Estruturas de repetição (for)
- Operador módulo (%)
- Organização básica de projeto

---

## ▶️ Como Executar

1. Clone o repositório:
   git clone https://github.com/kaioguarines/mini-sistema-java.git
2. Abra o projeto no IntelliJ IDEA.

3. Execute a classe principal (`Main`).

---

## 🎯 Objetivo do Projeto

Este projeto foi criado com o objetivo de fortalecer a base em lógica de programação utilizando Java, servindo como etapa inicial na jornada para desenvolvimento backend.
